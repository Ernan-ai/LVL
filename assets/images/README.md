# Logo Image

Place the octopus skull logo as `logo.png` in this directory.

The logo should be:
- Format: PNG with transparent background
- Recommended size: 512x512 pixels minimum
- The logo will be used in:
  - Splash screen (120x120)
  - Home page (80x80)
  - App icon (various sizes)

The logo will be tinted to match the app's color scheme automatically.
